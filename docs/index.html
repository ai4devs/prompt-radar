About the tool

Prompt feedback directly where developers write prompts.

Prompt Radar is designed for prompts-as-code. A developer selects a prompt in the editor, runs the analysis, and gets a score-driven view that highlights strengths, weaknesses, and actionable improvement suggestions inside VS Code.

01

Overall cleanliness score

A simple overall score helps developers quickly judge whether a prompt looks clean and ready or still needs revision.

02

Five category scores

The radar chart shows how the prompt performs across Security, Safety, Formatting, Efficiency, and Reliability.

03

Detailed feedback and assessment

Each category comes with an explanation and a lightweight user assessment so the tool can be reviewed and improved.

Scoring categories

Higher values in the radar mean better prompt quality.

Prompt Radar scores each dimension from weak to strong. A higher score means the prompt appears cleaner, more explicit, safer, and more robust in that category.

FormattingClear structure, explicit output expectations, and parseable responses.
ReliabilitySpecific, stable instructions that reduce ambiguity and inconsistent behavior.
EfficiencyConcise wording and useful context without unnecessary token cost.
SecurityReduced risk of leakage, unsafe access, injection, or sensitive exposure.
SafetyLower risk of harmful, unethical, biased, or policy-problematic outputs.

Workflow

Analyze, review, and refine prompts inside VS Code.

Select a prompt

The developer highlights a prompt, template, or message directly in the editor.

Run Prompt Radar

The extension analyzes the selection using the LLM-based scoring pipeline.

Review the score

The overall score and radar chart summarize how strong the prompt looks across the five categories.

Use the feedback

The developer reads the detailed analysis, gives quick assessment feedback, and improves the prompt.

Analysis view

A simple detailed analysis page for each prompt.

After analyzing a prompt, the extension can show the overall score, the radar chart, a short explanation, and a category-by-category assessment panel that lets users agree, disagree, or stay unsure.
